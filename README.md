# Brain Stroke Prediction Using PySpark

## Scalable Machine Learning for Healthcare Analytics

### 🎯 Project Overview
This project develops a predictive machine learning model to forecast brain stroke risk based on health indicators. Built with Python, scikit-learn, and PySpark.

### 📊 Dataset
- Patient health records
- Features: Age, Hypertension, Heart Disease, Glucose Level, BMI
- Target: Stroke / No Stroke

### 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- PySpark

### 📁 Project Structure

brain-stroke-prediction-pyspark/
├── README.md
├── requirements.txt
├── data/
├── src/
└── models/

### 🚀 How to Run

```bash
pip install -r requirements.txt
python src/model_trainer.py
```
